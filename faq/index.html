FAQ

Common questions

Short answers for people who want to improve their Twitch setup or understand what this site is for.

What is this website for?
This site collects Twitch-related resources I recommend or use myself: chat clients, emotes, browser extensions, moderation tools, overlays, event links, and small utility tools.
What should I install first for a better Twitch chat experience?
Start with Chatterino. It is usually the biggest quality-of-life upgrade if you watch Twitch often, especially when you care about emotes, readability, and faster chat interaction.
Why can’t I see some emotes in Twitch chat?
Many emotes are not native Twitch emotes. You usually need extensions or clients that support services like 7TV, BetterTTV, or FrankerFaceZ. The Emotes page explains what you need.
Do I need browser extensions if I already use Chatterino?
Not always. Chatterino improves the standalone chat experience, while browser extensions improve Twitch directly in your browser. If you use both Twitch in-browser and Chatterino, extensions can still be useful.
What is the Regex Generator for?
The Regex Generator helps moderators create patterns for blocking words with leetspeak, symbols, and spacing variations. It supports Chatty and Fossabot-style outputs.
What is Colors Cup?
Colors Cup is an inclusive esports event in Wiesbaden focused on fair play, respect, and community. I help with the project as an Event Coordinator. You can visit it at colors-cup.de.
Where can I see your moderator roles?
The Roles page shows selected Twitch partner channels I moderate for, including follower counts and mod since dates.
Can I use the overlays or tools for my own stream?
Yes, the resources are meant to help others improve their setup. Check the linked pages for instructions and always respect the original tool creators, licenses, and service rules.
How can I contact you?
The easiest way is through Discord: tukk7. You can also reach me through Twitch or X. The social icons at the top of this page link to my main profiles.

Still unsure where to start?

Begin with Better Chat, then add emote support and browser extensions. If you moderate, try the Regex Generator after that.
