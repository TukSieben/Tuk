7TV Emotes & Cosmetics

If Twitch chat sometimes looks like random words instead of emotes, you are probably missing 7TV support. This page explains what 7TV does and how to get started.

What is 7TV?

7TV is a Twitch extension focused on emotes and chat cosmetics. It lets you see the emotes people are using, supports animated emotes, and adds cosmetic options like name paints and animated profile pictures. It is popular because it makes chat feel more expressive and gives communities their own visual style.

Features

What 7TV adds

7TV is mainly about emotes, but the cosmetics around it are a big part of why people use it.

More Emotes

See the emotes everyone else is using instead of only seeing plain text like emote names in chat.

Animated Emotes

Many 7TV emotes are animated, which makes reactions in chat feel faster, funnier, and more alive.

Name Paints

Name paints can give usernames gradients, animations, and special effects so they stand out in chat.

Profile Pictures

7TV can also show animated profile pictures in supported chat environments.

Emote Sets

Streamers can manage sets of emotes for their channel, so the community has emotes that fit its own style.

Community Style

Emotes and paints help chats feel more personal, especially in communities that use a lot of inside jokes.

Setup

How to get 7TV working

You do not need a complicated setup. Install 7TV, refresh Twitch, and check if emotes start showing correctly.

Install 7TV

Go to 7tv.app and install the browser extension for your browser.

Open Twitch again

After installation, refresh Twitch or restart your browser. Emotes should start rendering in supported chats.

Use it with a chat client

If you use Chatterino7, make sure 7TV support is enabled there too. That gives you a better standalone chat experience.

For streamers

Streamers can create or manage a 7TV emote set for their channel, then add emotes that fit their community.

Important to know

7TV mainly works in desktop browser environments and supported chat clients. The default Twitch mobile app does not support browser extensions, so emotes and cosmetics may not appear there the same way.
